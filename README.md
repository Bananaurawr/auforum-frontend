# AUForum - Frontend

A mini StackOverflow-style forum application where students can ask questions and share knowledge.

🌐 **Live:** [ankuforum.web.app](https://ankuforum.web.app)

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router
- Axios
- Firebase Hosting

## Pages

- `/login` — User login
- `/register` — Create account
- `/home` — Browse all questions
- `/question/:id` — View question and answers
- `/ask` — Post a new question

## Deployment

Hosted on **Firebase Hosting**. Auto-deploys on every push to `main` branch.
